# Orca Projects
Implementations of basic logic gates, insertion sort, bubble sort, and bead sort in the Orca programming language and livecoding environment. Developed as a part of the [Pioneer Academics research program](https://pioneeracademics.com/), supervised by [Professor David Perkins](https://www.hamilton.edu/academics/our-faculty/directory/faculty-detail/david-perkins) of Hamilton College. Implemented collaboratively with [https://github.com/zeddybot](https://github.com/zeddybot)

To learn more about Orca, visit [here](https://github.com/hundredrabbits/Orca).
